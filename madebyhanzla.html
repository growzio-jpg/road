One-to-One (1:1)

One-to-Many (1:M)

Example:

First Normal Form (1NF)

‚ùå NOT in 1NF

ID Name Courses
101 John Math, Physics

Multiple values in one cell!

‚úì In 1NF

ID Name Course
101 John Math
101 John Physics

Atomic values only!

1NF Rules:
  • Each cell contains single value
  • No repeating groups
  • Unique column names
  • Each row is unique

Second Normal Form (2NF)

Rule: Must be in 1NF + No partial dependencies

Partial Dependency: When a non-key attribute depends on only PART of a composite primary key

‚ùå NOT in 2NF

Student_Course Table
PK: (StudentID, CourseID)

StudentName depends only on StudentID
CourseName depends only on CourseID
‚ö†Ô∏è Partial dependencies exist!

‚úì In 2NF

Split into 3 tables:

Student(StudentID, Name)
Course(CourseID, Name)
Enrollment(StudentID, CourseID, Grade)

‚úì No partial dependencies!

Third Normal Form (3NF)

Rule: Must be in 2NF + No transitive dependencies

Transitive Dependency: A ‚Üí B and B ‚Üí C, then A ‚Üí C

‚ùå NOT in 3NF

Student(StudentID, Name, Dept, DeptHead)

StudentID ‚Üí Dept
Dept ‚Üí DeptHead
‚ö†Ô∏è Transitive: StudentID ‚Üí DeptHead

‚úì In 3NF

Student(StudentID, Name, Dept)
Department(Dept, DeptHead)

‚úì No transitive dependencies!

Functional Dependency

Definition: If A ‚Üí B (A determines B), then for each value of A, there's only ONE corresponding value of B.
Example: StudentID ‚Üí StudentName
Each StudentID has exactly ONE name

Compiler vs Interpreter

üîß Compiler

  • Translates entire program at once
  • Creates executable file
  • Faster execution
  • Shows all errors together
  • Example: C, C++

üîÑ Interpreter

  • Translates line by line
  • No executable file
  • Slower execution
  • Stops at first error
  • Example: Python, JavaScript

get() vs getline()

get(): Reads until newline but LEAVES '\n' in buffer
getline(): Reads until newline and REMOVES '\n' from buffer

cin.get(name, 50); // Leaves newline
cin.getline(name, 50); // Removes newline

= (Assignment)

Assigns value to variable

== (Equal To)

Compares two values

‚ö†Ô∏è Common Mistake:
